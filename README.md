# House Price Prediction Analysis

This project is a simple **Data Analysis and Machine Learning application** with a graphical user interface (GUI) built using **Tkinter**.
It allows users to load a dataset, clean it, and apply multiple regression models to predict house prices.

---

# Features

*  Easy dataset loading (CSV format)
*  Data preprocessing:

  * Handling missing values
  * Removing outliers using IQR method
*  Apply multiple regression models:

  * Linear Regression
  * Lasso Regression
  * Ridge Regression
*  Model evaluation using:

  * Mean Squared Error (MSE)
  * Mean Absolute Error (MAE)
*  Visualization of actual vs predicted values
*  Simple and user-friendly GUI using Tkinter

---

# Models Used

* Linear Regression
* Lasso Regression
* Ridge Regression

---

# Dataset Requirements

The dataset should include the following columns:

* Median_House_Value (Target)
* Median_Income
* Median_Age
* Tot_Rooms
* Tot_Bedrooms
* Population
* Households
* Latitude
* Longitude
* Distance_to_coast
* Distance_to_LA
* Distance_to_SanDiego
* Distance_to_SanJose
* Distance_to_SanFrancisco

# How to Use

1. Click on **"Load Dataset"** and select your CSV file
2. Click on **"Run Analysis"**
3. The application will display:

   * Model evaluation metrics
   * Visualizations
